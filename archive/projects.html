Save the World
Gaming technologies and simulations

In my third year of university, I was taught core game technologies such as collision detection and response, pathfinding and AI behaviour. We were then given a fairly open-ended coursework as a change to show off what we'd learnt using the Unity game engine. My solution was "Save the World", a short boss-battle heavily inspired by Undertale.

  • Appropriate use of collision response and feedback
  • Use of state-based behaviour
  • A gameplay experience with win or lose conditions
  • The ability to restart gameplay
Comparing Different Techniques in Formation Pathfinding
Undergraduate major dissertation project

For the end of my third year at Newcastle University, I decided to write my dissertation on the comparison of various pathfinding techniques used when groups of entities need to move together. I compared their efficiency and realism between different game environments to provide recommended implementations. To achieve this, I wrote a configurable formation pathfinding algorithm in C# and designed a dynamic visual implementation environment using Unity.

Programmatic objectives met:
  • Abstraction so the algorithm only uses very basic interfaces
  • Configuration to allow for testing between all combinations of features in Unity
  • Visualisation of each step taken by the entities
  • Configuration of the environment's grid density, obstacle concentration and entity frequency
