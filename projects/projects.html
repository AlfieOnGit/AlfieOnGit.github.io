Conway's Game of Life
C++ programming introduction

At the beginning of my master's year, my lecturers gave us a crash course in C++ programming. This project was the resulting coursework, intended to show off our understanding of C++ features. It was written in pure C++ for a command line interface.

Features included:
  • Object orientated programming
  • Appropriate use of inheritance
  • Appropriate use of pointers and references
  • Operator overloading
  • Templates
  • Threading
Alien Planet
Graphics programming using OpenGL

For our second module, we were taught about graphics programming. For this coursework, I made a 3D scene of an alien planet using OpenGL and C++.

Features included:
  • Textures and meshes
  • Terrain rendering from height maps
  • MVP vertex transformations
  • Light handling
  • Reflections using environment mapping
  • Scene graph system
  • Shadows
One Cute Tabby, Not Too Shabby
Game technologies

For my third and final taught content module, we were taught about how game engines implement core game technologies, such as physics, AI and networking. Our coursework was to take inspiration from the game "Little Kitty, Big City" and design a basic game that included such technologies.

Features included:
  • CMake usage
  • Collision detection and resolution
  • Application of forces
  • Function pointer AI behaviour
  • State machines
  • Screen management using pushdown automata
