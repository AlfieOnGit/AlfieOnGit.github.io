Vulkan Engine 2025
Current personal project

In university, I was taught rendering using OpenGL. However, in my free time, I decided to investigate using Vulkan. I had heard it was "a fast but hard GPU API library" and, as a programmer with a little too much self-confidence, I decided to start writing a Vulkan graphics engine.

This engine is my current project, so I don't have much to show off just yet, but do ask me about it if you're interested!
OpenGL Alien Planet 2024
Graphics programming using OpenGL

For our second module, we were taught about graphics programming. For this coursework, I made a 3D scene of an alien planet using OpenGL and C++.

Features included:
  • Textures and meshes
  • Terrain rendering from height maps
  • MVP vertex transformations
  • Light handling
  • Reflections using environment mapping
  • Scene graph system
  • Shadows
Unity Alien Planet 2023
Graphics for Games

During the undergraduate stage of my degree, I was taught the basics of the rendering pipeline. For the coursework, we used a variety of rendering techniques in Unity to produce an alien planet scene. The main focus of this coursework was to include as many features as possible so ignore how unrealistic it looks!

Unity features included:
  • A day-night cycle lit by dynamic lighting
  • Scene hierarchy, used to created orbiting objects
  • Terrain and multiple biomes
  • Fog
  • Consistent framerate
